# Watchlists

Watchlists sind die benannten Sammlungen von Symbolen, die du aktiv verfolgst. Jede andere Flaeche (Scanner, Alerts, Paper-Trading-Empfehlungen) wird durch die aktive Watchlist gefiltert.

## Was du siehst

- **Eine Karte pro Watchlist** mit ihren Symbolen, Namen und Tags. Die Default-Seed-Watchlists ("Tech Giants", "Crypto Proxies") werden beim ersten Login angelegt.
- **Add-Item-Formular** am unteren Rand jeder Karte — Symbol, optionaler menschen-lesbarer Name, kommagetrennte Tags.

## Symbole

- **Aktien** nutzen einfache Ticker (AAPL, MSFT, NVDA).
- **ETFs** nutzen ebenfalls einfache Ticker (VOO, SPY, XLK).
- **Crypto** nutzt die Slash-Form (BTC/USD, ETH/USD).
- **Nicht-US-Aktien** nutzen das Exchange-Suffix (SAP.DE, BMW.DE, LVMH.PA, 7203.T).

## Tags

Tags sind freie Text-Labels, die du an einem Watchlist-Eintrag haengst. Alert-Regeln koennen auf Tags matchen ("alles mit Tag 'priority' soll bei negativen News feuern"), daher sind sie eine nuetzliche Organisations-Schicht, sobald deine Watchlist groesser wird.

## Wie es persistiert

Watchlist-Daten liegen in der App-Datenbank und sind von jedem Backup, Export und Import abgedeckt.
